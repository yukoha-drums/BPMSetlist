# BPM Setlist

iOS 向けの **BPM セットリスト + メトロノーム** アプリです。SwiftUI と SwiftData を使用しています。

## 要件

- Xcode（iOS 17.5 以上向けにビルド）
- 実機またはシミュレータ

## ドキュメント

機能追加や構成の理解には、次を参照してください。

- **[開発ガイド（アーキテクチャ・拡張ポイント）](docs/DEVELOPMENT.md)**

## ビルド

1. `BPMSetlist.xcodeproj` を Xcode で開く
2. スキーム `BPMSetlist` を選び Run（⌘R）
