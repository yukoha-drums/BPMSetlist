# 🎯 最も簡単な方法 - Xcode手動実行

App Storeの差し戻し対応：iPhone版とiPad版のスクリーンショットを分けて生成

---

## 📱 手順（10分で完了）

### ステップ1: Xcodeでプロジェクトを開く

```bash
open /Users/yuichiro.kohata/00.work/codes/BPMSetlist/BPMSetlist.xcodeproj
```

### ステップ2: iPhoneでスクリーンショットを撮影

1. **シミュレーターを選択**
   - Xcode上部のデバイス選択ドロップダウンをクリック
   - **iPhone 15 Pro Max** を選択
   
2. **アプリを実行**
   - `⌘R` または Play ボタンをクリック
   - アプリが起動するのを待つ
   
3. **スクリーンショットを撮影**（以下の5画面）
   
   **画面1: メイン画面（空）**
   - アプリが開いたらそのまま
   - `⌘S` または File > Save Screen
   
   **画面2: 曲を追加**
   - 「+」ボタンをタップ
   - 曲情報を入力（例: Shape of You, Ed Sheeran, 96 BPM）
   - 保存
   - `⌘S` でスクリーンショット
   
   **画面3: 曲追加画面**
   - 再度「+」ボタンをタップ
   - `⌘S` でスクリーンショット
   - キャンセルで閉じる
   
   **画面4: 再生中**
   - 再生ボタンをタップ
   - `⌘S` でスクリーンショット
   
   **画面5: 設定画面**
   - 設定ボタンをタップ
   - `⌘S` でスクリーンショット

### ステップ3: iPadでスクリーンショットを撮影

1. **シミュレーターを変更**
   - Xcode上部のデバイス選択で **iPad Pro (12.9-inch)** を選択
   
2. **ステップ2と同じ操作を繰り返す**
   - アプリを実行（⌘R）
   - 5つの画面でスクリーンショットを撮影（⌘S）

### ステップ4: スクリーンショットを整理

スクリーンショットはデスクトップに保存されます：

```
~/Desktop/
├── スクリーンショット 2026-02-10 8.00.00.png  (iPhone)
├── スクリーンショット 2026-02-10 8.01.00.png  (iPhone)
...
├── スクリーンショット 2026-02-10 8.05.00.png  (iPad)
├── スクリーンショット 2026-02-10 8.06.00.png  (iPad)
...
```

**整理方法:**

1. デスクトップに2つのフォルダを作成
   ```
   ~/Desktop/AppStore-iPhone/
   ~/Desktop/AppStore-iPad/
   ```

2. スクリーンショットをドラッグ&ドロップで分類
   - iPhone用: 最初の5枚
   - iPad用: 次の5枚

3. ファイル名をリネーム（推奨）
   ```
   01-main-empty.png
   02-main-with-songs.png
   03-add-song.png
   04-playing.png
   05-settings.png
   ```

### ステップ5: App Store Connectにアップロード

1. **App Store Connect** にアクセス
   https://appstoreconnect.apple.com/

2. **アプリを選択** → **1.0 準備中** → **スクリーンショット**

3. **"View All Sizes in Media Manager"** をクリック

4. **iPhone用セクション**
   - `~/Desktop/AppStore-iPhone/` から5枚をドラッグ&ドロップ
   - 6.7インチ（iPhone 15 Pro Max）のスロットに配置

5. **iPad用セクション**
   - `~/Desktop/AppStore-iPad/` から5枚をドラッグ&ドロップ
   - 12.9インチ（iPad Pro）のスロットに配置

6. **保存** → **Submit for Review**

## ✅ 重要ポイント

- ✅ **必ず別々のシミュレーターで撮影**
  - iPhone: iPhone 15 Pro Max
  - iPad: iPad Pro (12.9-inch)

- ✅ **iPadのスクリーンショットにiPhoneのフレームが含まれないこと**

  1. メイン画面（空）

  3. 曲追加画面

  5. 設定画面

## 💡 ヒント

### スクリーンショットの撮り方

**方法1: ⌘S**
- シミュレーターをアクティブにして `⌘S`
- デスクトップに自動保存

**方法2: Edit > Copy Screen**
- Edit > Copy Screen
- プレビューアプリで新規作成 → ペースト → 保存

### 画面サイズの確認

- iPhone: 縦長（約2:1の比率）
- iPad: 正方形に近い（約4:3の比率）

ファイル名に「iPhone」「iPad」が含まれていることを確認！
